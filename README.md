# NG-SpringMongo 
Java Spring-boot, Maven, MongoDB And Angular CLI TO DO Apps



> ### REQUIREMENT =
      - Java SE 8 +
      - MAVEN
      - Spring-Boot Tool
      - MongoDB
      - Angular CLI 5 +
      
> ### FOLDER ROOT =
      - Backend     : todoapp folder
      - Frontend    : angular-frontend
> ### How To Run =
     >> Backend:
        - Assuming Maven And All Requirement Have been Install
        - open cmd / terminal 
        - go to directory todoapp in cmd
        - execute command mvn spring-boot:run
        - If Success, default localhost using port : 8080
    >> Frontend
        - Assuming Angular CLI have been installed
        - open cmd or terminal command line
        - go to directory angular-frontend
        - npm install to install it's all dependencies
        - run command "ng serve --open"
        - if success, will auto open default browser that link to localhost: 4200
